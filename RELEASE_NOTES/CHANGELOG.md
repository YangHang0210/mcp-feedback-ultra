# 更新日志

## [2.6.9] - 2026-04-27

### 说明

本项目基于 [noopstudios/interactive-feedback-mcp](https://github.com/noopstudios/interactive-feedback-mcp) 初始化并重构，感谢原作者的贡献。

### 特性

- 支持 Web UI 和桌面应用双界面模式
- 自动识别 SSH Remote、WSL 等特殊环境
- 交互式反馈收集功能
- 跨平台支持（Windows、macOS、Linux）
- 支持图片上传和 Markdown 渲染
- 会话管理和历史记录

### 技术栈

- Python 3.11+
- FastMCP
- FastAPI + WebSocket
- Tauri (桌面应用)
